# Unsan Academy Job Database

자동차 애프터마켓 산업의 모든 커리어 경로를 정의하는 직업 데이터베이스입니다.
이 리스트는 플랫폼의 **'검색 키워드'**이자, 스킬 트리의 **'노드(Node)'**가 됩니다.

---

## 1. The 5 Core Stats (자동차 직무 5대 역량)

모든 직업은 이 5가지 능력치의 조합으로 정의됩니다.

| Stat | 이름 | 설명 |
|------|------|------|
| **[T]** | Tech (기술/지능) | 원리를 이해하고, 데이터(회로도, 파형)를 분석하여 진단하는 능력 (뇌) |
| **[H]** | Hand (손기술/숙련도) | 도구를 능숙하게 다루고, 빠르고 정밀하게 시공하는 능력 (손) |
| **[A]** | Art (예술/감각) | 미적 감각, 디테일, 색감, 마감 품질을 보는 눈 (눈) |
| **[B]** | Biz (경영/영업) | 고객 설득, 매출 창출, 조직 관리, 마케팅 능력 (입) |
| **[O]** | Ops (운영/규정) | 매뉴얼(SOP) 준수, 안전 관리, 법적 절차 이행 능력 (습관) |

## 4. 변경 이력

### v2.0 (2024)
- 100개 → 88개로 통합 정리
- 통합된 직업:
  - 퀵-메카닉 + 배터리/전장맨 → 경정비 테크니션
  - 인젝터/고압펌프 + 터보차저 수리 → 디젤 스페셜리스트
  - 오토미션 + 수동미션/클러치 → 변속기 전문가
  - 엔진 보링/오버홀 + 타이밍벨트/체인 → 엔진 오버홀 마스터
  - 디테일러 + 광택사 + 코팅 + 헤드라이트 복원 → 디테일링 전문가
  - 유리 복원 + 유리 교환 → 자동차 유리 전문가
  - 가죽시트 + 실내파츠 복원 → 실내 복원 전문가
  - 실내 랩핑 + 천장 엠보싱 → 실내 커스터마이징 전문가
  - 카오디오 + 방음/방진 → 사운드 시스템 전문가
  - 캠핑카 빌더 + 캠핑카 전기 → 캠핑카 빌더
  - 출장 세차 + 출장 오일/배터리 → 출장 서비스 전문가
- 테슬라 공인 바디샵, 하이브리드 전문가 등 위치 재배치
